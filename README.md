# V2free自动签到

参考学习 [V2free-Checkin](https://github.com/founder-yu/V2free-Checkin.git)

### 使用教程

1. 点击Fork按钮将项目Fork到自己仓库

2. 配置参数

    依次点击上栏【Setting】->【Security】->【Secrets】->【Actions】->【New repository secrets】 添加账号和密码，示例如下：

    |key |value                     |
    |:--:|:------------------------:|
    |USER|123@qq.com,567@outlook.com|
    |PWD |ccc,aaa                   |
